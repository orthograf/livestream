LiveStream plugin for AltoCMS
=================

Описание
---------
Замена блока "Прямой эфир" на блок с выводом Активности

-   Идея: LSP-GoStream (goweb.pro)
-   Модернизация и адаптация плагина для AltoCMS: Orthograf
-   Для связи: support@orthograf.ru
